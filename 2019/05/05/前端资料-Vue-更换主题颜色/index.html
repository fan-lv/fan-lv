更换主题颜色
发表于 2019-05-05 | 更新于 2021-07-06
前端资料Vue
Vue
  1. 把 css 文件放到服务器上，通过 XMLHttpRequest 请求文件，获取到文件内容之后替换颜色。
  2. 通过 getStyleTemplate 方法，把要更改的颜色替换成临时变量字段，如”#001528”颜色先替换成”primary”字段
  3. 再定义一个颜色变量，为需要替换的新颜色。把替换的变量字段最终替换成新的颜色
