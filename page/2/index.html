页面加载过程

  • 浏览器根据 DNS 服务器查询得到域名对应的 IP 地址
  • 向这个 IP 机器发送 HTTP 请求
  • 服务器收到、处理并返回 HTTP 请求
  • 浏览器得到返回的内容

例如访问 GitHub，会解析得到 IP 地址为 52.74.223.119，浏览器会向这个地址发送 HTTP 请求。
